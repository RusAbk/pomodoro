# 🍅 Pomodoro Timer

Beautiful and productive Pomodoro Timer built with Flutter. Focus better, work smarter!

## ✨ Features

- 🍅 **Classic Pomodoro Technique** - 25min work, 5min break, 15min long break
- 🎨 **Beautiful UI** - Glassmorphism design with smooth animations
- 📊 **Statistics Tracking** - Monitor your productivity and streaks
- ⏱️ **Flexible Timing** - Switch between work/short/long break modes
- 🔄 **Smart Transitions** - Automatic or manual mode switching
- 📱 **Responsive Design** - Works on web, mobile, and desktop
- 🌐 **PWA Ready** - Install on any device like a native app

## 🛠️ Development

```bash
# Clone the repository
git clone https://github.com/RusAbk/pomodoro.git
cd pomodoro

# Get dependencies
flutter pub get

# Run the app
flutter run
```
